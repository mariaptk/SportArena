# SportArena — Информационный портал спортивных событий

## Описание проекта
**SportArena** — это современный веб-интерфейс для поиска спортивных мероприятий и бронирования отелей. Проект ориентирован на атлетов и болельщиков, предоставляя единую точку доступа к расписанию соревнований и сопутствующей инфраструктуре.

Данный проект реализован в рамках учебной дисциплины как результат освоения полного цикла настройки рабочего окружения: от проектирования в Figma до контроля версий в Git.

---

## Дизайн и концепция
Проект выполнен в минималистичном стиле с использованием контрастной темной темы, что подчеркивает премиальность и энергичность спортивной тематики.

* **Мудборд:** Эстетика профессионального спорта, динамика, четкие линии.
* **Типографика:** * Заголовки: шрифт *Cinzel* (придает монументальность).
    * Текст и навигация: *Source Code Pro* (акцент на технологичность).
* **Цветовая палитра:** Глубокий черный (#000), чистый белый (#FFF) и полупрозрачные оверлеи для создания глубины.

## Структура репозитория
* `/css` — каскадные таблицы стилей (основной файл `style.css`).
* `/js` — скрипты для обеспечения интерактивности (`script.js`).
* `/images` — оптимизированные графические ресурсы проекта.
* `/design` — документация по проектированию и ссылка на прототип.
* `index.html` — точка входа, главная страница портала.

---

## Технический стек и инструменты
Проект использует современный инструментарий фронтенд-разработки:
* **VS Code** + расширения (Live Server, Prettier) для написания и форматирования кода.
* **Git** для фиксации изменений и управления историей разработки.
* **Node.js & npm** для управления зависимостями (установлены пакеты `lodash`, `nodemon`).
* **Markdown** для ведения технической документации.

---

## Как запустить проект
1.  Убедитесь, что у вас установлен **Node.js**.
2.  Склонируйте репозиторий.
3.  Выполните команду `npm install` для установки окружения.
4.  Запустите файл `index.html` через расширение **Live Server** в VS Code.
